## Getting Started

1. Verify that version 16.14.0 of node.js (https://nodejs.org/en/) is installed.

2. Install dependencies:

```bash
yarn install
```

3. Start development server:

```bash
yarn run dev
```

## Running Tests

1. Start test server:

```bash
yarn run server
```

2. In the Cypress IDE, choose which tests to run.
